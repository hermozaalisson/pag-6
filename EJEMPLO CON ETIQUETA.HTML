Etiquetas HTML

Este ejemplo muestra como combinar algunas de las etiquetas más básicas de HTML5.
. Recuerda que es importante entender las diferencias entre ellas

Etiquetas ul + il

Si listamos personas nominadas a los Oscars, dado que el orden no altera el significa, debemos usar ul

  • David Verdaquer
  • Jesús Castro
  • Israel Elejalde
  • Dani Rovira

Etiqueta ol + il

En el caso de que estemos listando elementos donde el orden es importante, como por ejemplo la tabla de posiciones de la COPA AMERICA 2019 , debemos usar ol

  1. Brasil
  2. Peru
  3. Argentina
  4. Chile
